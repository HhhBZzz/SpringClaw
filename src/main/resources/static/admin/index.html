05 / Scheduled Work

定时任务

这里管理任务底座。任务可以定时执行某个 skill，也可以定时发一条完整 Agent prompt；执行记录默认独立归档，可选写回会话或投递到飞书。

-任务总数
-已启用任务
-已停用任务
-最近一次状态

手动创建

适合直接按字段配置。`targetType=skill` 时填 skillId；`targetType=agent` 时 `targetRef` 固定填 `prompt`，正文写在 `inputPayload`。

如果你只是想跑一个普通 Agent 问句，推荐 `targetType=agent` + `targetRef=prompt`。
